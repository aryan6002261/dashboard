# 📊 Dashboard Layout - ABC Company

A responsive **dashboard layout UI** built using HTML and CSS Grid/Flexbox. It simulates a basic company admin panel with a header, navigation sidebar, and main content area.

---

## 🖼️ Preview Layout

The dashboard is divided into three main sections:

- **Header**
- **Navigation Sidebar**
- **Main Content Area**

---

## 🚀 Features

- 🔍 Search bar in header
- 🏢 Company logo + branding
- 🔐 Login & Sign Up buttons
- 📂 Sidebar navigation menu
- 📊 Structured dashboard content
- 🧩 Clean grid-based layout structure

## 🧭 Sections Overview

### 🟦 Header
Contains:
- Search bar
- Company logo and name (ABC Company)
- Authentication buttons (Login / Sign Up)

### 🟩 Navigation Sidebar
Includes links:
- Home
- Profile
- Settings
- Community
- Support
- Contact

### 🟨 Main Content
Displays:
- Welcome message
- Company overview
- Today's statistics
- Recent activity updates
- Upcoming deadlines

## ▶️ How to Run

1. Clone or download the project
2. Make sure `index.html` and `styles.css` are in the same folder
3. Open `index.html` in any modern web browser
